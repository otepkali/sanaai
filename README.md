# Налоговый калькулятор Казахстана 2026

Веб-инструмент для бухгалтеров и владельцев ИП: быстрый расчёт налогов с прозрачной
разбивкой каждой суммы — не только итог, но и из чего он сложился.

Разработан для **Sana Ai**.

## Модули

- **ФОТ** (форма 200) — ОПВ, ВОСМС, ИПН, «на руки» и полная стоимость работника для
  работодателя (СО, ООСМС, ОПВР, СН), с учётом льготных категорий.
- **Упрощённая декларация** (форма 910) — налог по ставке маслихата, льгота за работников,
  проверка лимита дохода 600 000 МРП/год.
- **НДС** — начисление «сверху» и выделение «в том числе», проверка порога обязательной
  постановки на учёт (10 000 МРП).
- **Сравнение режимов** — упрощёнка vs ОУР бок о бок, с наглядным выводом, что выгоднее.

## Технологии

- [Next.js 16](https://nextjs.org) (App Router) + TypeScript
- Tailwind CSS v3 + [shadcn/ui](https://ui.shadcn.com)
- [Zod](https://zod.dev) — валидация ввода
- [Vitest](https://vitest.dev) — юнит-тесты налоговой логики

## Архитектура

Вся налоговая логика — чистые функции в `lib/`, без привязки к UI:

```
lib/
  tax-config-2026.ts   — все ставки и константы 2026 года (единственный источник истины)
  payroll.ts           — расчёт ФОТ / зарплаты
  simplified.ts        — упрощённая декларация (910)
  vat.ts               — НДС
  compare.ts           — сравнение режимов
  schemas.ts           — Zod-схемы валидации форм
  format.ts            — форматирование сумм и процентов
  __tests__/           — юнит-тесты на эталонных примерах
components/
  calculators/         — UI каждого калькулятора
  ResultBreakdown.tsx  — компонент детальной разбивки результата
```

> Ставки социальных платежей содержат краевые случаи (льготные категории, пределы баз) и
> периодически меняются. Это инструмент-помощник, а не замена бухгалтеру — см.
> предупреждение в начале `lib/tax-config-2026.ts`.

## Запуск локально

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Тесты


## Деплой на Vercel

1. Инициализируйте git-репозиторий и запушьте проект в GitHub:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin <URL вашего репозитория>
   git push -u origin main
   ```
2. На [vercel.com](https://vercel.com/new) нажмите **Import Project** и выберите этот
   репозиторий.
3. Vercel автоматически распознает Next.js — никаких дополнительных настроек и переменных
   окружения не требуется. Нажмите **Deploy**.
4. После сборки проект будет доступен по выданному адресу `*.vercel.app`; при пуше в `main`
   деплой обновляется автоматически.
